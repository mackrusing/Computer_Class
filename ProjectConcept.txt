COMMANDS

help:

  opens help menu


list category, list cat, ls category, ls cat:

  lists all categories


list {category_name}, ls {category_name}:

  lists all items in category


list all, ls all:

  lists all item names


info {item_name}:

  item name
  item price
  item category
  item description


view {item_name}:

  opens image of product in web browser
